# Deadliner 隐私政策（版本 4.0.0）

**运营主体：周擎**  
**更新日期：2025-12-2**

本隐私政策用于向您说明 Deadliner（以下简称“本应用”）在提供服务过程中如何收集、使用、共享、保存及保护您的个人信息。  
请您在使用前认真阅读。若您不同意本政策任何内容，请停止使用。

本应用为个人开发工具类应用，不要求注册账号，不收集任何可识别用户身份的信息（如姓名、手机号、身份证号、邮箱等）。

---

# 一、适用范围
本政策适用于 Deadliner Android 客户端，包括任务提醒、习惯管理、智能文本分析（Deadliner AI）、WebDAV 同步等全部功能。

---

# 二、我们如何收集和使用您的个人信息

本应用遵循“最小必要原则”，仅在您使用功能时收集或使用信息，且不收集可识别您身份的任何信息。

## 2.1 本地任务与习惯管理
- **用途**：创建、保存任务和习惯
- **收集内容**：任务内容、时间、提醒设置（仅本地存储）
- **必要性**：必要

## 2.2 日历写入（可选）
- **用途**：在本机日历中写入任务事件
- **权限**：日历读写
- **必要性**：可选，启用时必需

## 2.3 通知提醒（必要）
- **用途**：向您推送任务提醒
- **权限**：通知、前台服务、闹钟
- **必要性**：必要
- **拒绝影响**：将无法收到提醒

## 2.4 Deadliner AI（可选）
- **用途**：文本任务提取与结构化分析
- **收集内容**：仅包括您主动输入并提交至模型服务的文本
- **API Key**：由您填写，使用 Android Keystore + AES/GCM 本地加密保存
- **传输链路**：
  - 默认：设备 → 开发者服务器（仅转发，不存储） → DeepSeek
  - 或您自行填写的 OpenAI-Compatible API
- **必要性**：可选，不使用则无需提交任何数据

## 2.5 读取剪贴板（可关闭）
- **用途**：在您打开 AI 输入界面时，自动识别剪贴板内容以辅助任务提取
- **机制**：不会在后台持续读取，仅在使用 AI 功能时读取
- **必要性**：可选

## 2.6 WebDAV 同步（可选）
- **用途**：跨设备同步任务数据
- **上传内容**：本地数据库快照
- **账号密码**：明文保存在本机 SharedPreferences（风险自行承担）
- **必要性**：可选

## 2.7 更新检查（必要）
- **用途**：向 GitHub 查询版本更新
- **收集内容**：仅访问公开版本信息，无任何用户数据上传
- **必要性**：必要

---

# 三、个人信息的存储

## 3.1 存储地点
- 本地任务与习惯数据：存储于设备本地
- WebDAV 数据：存储于您选择的服务器
- API Key：加密后本地存储
- AI 请求内容：仅实时传输，不在服务器落盘

## 3.2 保留期限
- 本地数据：直至用户删除或卸载应用
- WebDAV 数据：由用户自行控制
- API Key：由用户自行删除
- AI 请求内容：不保存

---

# 四、是否进行用户画像
本应用不进行用户画像、不开展广告定位、不做行为分析。

---

# 五、Cookie 和同类技术
本应用不使用 Cookie、Web Beacon、Flash Cookie 等自动数据收集技术。  
仅使用本地 SQLite 与 SharedPreferences 保存本地数据和设置。

---

# 六、我们如何共享、转让、公开披露您的个人信息

## 6.1 共享
本应用默认不共享您的任何信息。  
仅以下场景由您主动触发：
1. 使用 Deadliner AI 时，文本发送至模型服务提供方（DeepSeek 或您自定义的 API）。
2. 使用 WebDAV 同步时，任务快照上传至您配置的服务器。

## 6.2 转让
不进行任何用户信息转让。

## 6.3 公开披露
不公开披露任何用户信息。

## 6.4 法定例外
在以下情形可能无需征得您的授权：
- 涉及国家安全、公共安全事项
- 按法律法规要求向行政或司法机关提供
- 为保护您或他人的生命财产安全
- 为保障系统安全性所必需的审计行为

---

# 七、我们如何保护您的信息

## 7.1 技术措施
- 使用 Android Keystore + AES/GCM 加密存储 API Key
- 全部网络通信采用 HTTPS
- 不含广告 SDK、不含行为分析 SDK
- 不含任何后台自动上传行为

## 7.2 风险提示
即使我们采取了合理措施，仍可能因：
- 恶意软件
- 设备 Root
- WebDAV 服务器遭攻击  
  导致数据泄露风险，请您注意加强设备和服务器安全。

## 7.3 安全事件响应
若发生信息安全事件，我们将通过应用内通知或 GitHub 公告告知并采取补救措施。

---

# 八、您如何管理个人信息

## 8.1 访问、修改、删除
- 可在应用内修改任务与习惯
- 可删除 WebDAV 配置与数据
- 可删除 API Key
- 卸载应用可删除全部本地数据

## 8.2 撤回授权
用户可在 Android 系统设置中关闭：
- 通知权限
- 网络权限
- 剪贴板权限
- 日历权限  
  关闭后对应功能将受影响。

## 8.3 禁用 Deadliner AI
不填写 API Key 即可完全禁用 AI 功能。

---

# 九、未成年人保护
本应用适用于 **12 岁及以上用户**。  
我们不主动收集未成年人的信息。  
监护人如发现未成年人误向本应用提交信息，可联系我们删除。
